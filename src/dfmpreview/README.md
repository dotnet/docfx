## Introduction
This is a DFM preview vscode extension which you can get from the [DocFX Flavored Markdown](http://dotnet.github.io/docfx/spec/docfx_flavored_markdown.html)

## Features
- Live preview   
  The shortcuts are
    - `ctrl+shift+q`    Preview
    - `ctrl+k q`        Preview to Side
