Class CodeSnippetParser

Match code snippet from markdown files. Code snippet syntax: {{"relativePath/sourceFilename"}} or {{"relativePath/sourceFilename"[startline-endline]}} where starline and endline should be integers and are both optional, the default values for which are 1 and the fileLength.
