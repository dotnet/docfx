# API Reference

This file is in the `reference/` folder which has `auto: false` in its `toc.yml`.

This means:
- Files are **not** auto-added to the TOC
- Each file must be manually listed in `toc.yml`
- You have full control over the TOC structure

## When to use auto: false

- When you need a specific order
- When you want to exclude certain files
- When you need custom names for TOC entries
